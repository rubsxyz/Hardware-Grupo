O que são os Barramentos?

Um barramento(bus), é uma caminho comun por onde passam os dados em um computaor.

O tamanho do barramento é importante pois determina quantos dados serão transmitidos de uma só vez.

Tipos de Barramentos.

Barramentos Internos

É o tipo de barramentos que ligam a CPU aos equipamento que ficam dentro do gabinete.

  1. Barramento IDE
    Serve para ligar unidades de armazenamento internas a motherboard(é usado um cablo flat nestas ligações);
  2. Barramento ISA
    É mais comum em computador antigos, é usado para as placas de expansão(placas de vídeo, placas de som);
  3. Barramento PCI
    É o substituto dos Barramentos ISA;
  4. Barramento AGP
    É um barramento apenas utilizado para placas de vídeo;
  5. Barramento SCSI
    É um barramento apenas utilizado em servidores que permite uma maior velocidade de coneção dos discos;

Barramentos Externos

Portas ou Interfaces.

  1. Porta PS/2
    É usada para ligar o teclado e o rato, e só existiam duas na traseira do computador;
  2. Porta Séria
    É usada para equipamentos que transferem pouca informação(ratos, cameras e outros equipamentos);
  3. Porta Paralela
    É usada para ligar impressoras;
  4. Portas USB
    É a mais usada atualmente, substitui as portas faladas anteriormente, e ainda tem mais funções como a de carregar equipamentos.;
  5. PCMCIA
    É usado para ligar placas e discos externos(Placa de som, placa de rede, discord portateis);
